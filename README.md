# Blast Radius

This repository contains the current public artifacts for the Blast Radius research project.

## Contents

### Measuring Trust Concentration as an Architectural Property
The research paper describing the motivation, framework, methodology, limitations, and future direction of the project.

### Blast Radius Reference Instrument
The companion scoring workbook implementing the current version of the framework and its evaluation criteria.

## Purpose

Blast Radius explores whether trust concentration can be treated as an architectural property that can be measured independently of vulnerabilities, configuration, or compliance. The project is an ongoing research effort, and the framework will continue to evolve through independent validation and community feedback.

Questions, critiques, counterexamples, and independent scoring results are welcome.
